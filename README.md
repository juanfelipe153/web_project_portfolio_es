# Portafolio Personal – Juan Felipe Ramírez

Sitio web de portafolio personal desarrollado para presentar mi perfil profesional, habilidades técnicas y proyectos destacados como desarrollador web junior.

## Descripción

Este proyecto consiste en una página web responsiva diseñada para mostrar información profesional de forma clara y moderna. Incluye una presentación personal, listado de tecnologías, proyectos destacados y enlaces de contacto.

## Link a la página

https://juanfelipe153.github.io/web_project_potfolio_es/

## Características

- Diseño responsivo para diferentes tamaños de pantalla.
- Estructura semántica con HTML5.
- Estilos personalizados con CSS3.
- Uso de Flexbox y Grid Layout.
- Tipografías personalizadas mediante `@font-face`.
- Sección de proyectos con enlaces a demo y repositorio.
- Información de contacto y redes sociales.

## Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Git
- GitHub
